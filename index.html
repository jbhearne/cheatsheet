CSS Cheat Sheet

Selector Terms
Selector Format Example Description
Universal * * {color: red;} Selects all elements
Type (Tag) tag h1 {color: red;} Selects all elements with the specified tag
Type (Atribute) [] [href] {color: red} Selects all elements that have the atribute specified
Type (Pseudo) : a:active {color: red;} Selects standard alternate states of elements
Class . .classname {color: red;} Selects all elements with specified class
ID # #IDname {color: red;} Selects the element with the specified ID

Combination Selectors
Combination Format Example Description
Multiple , h1, h2 {color: red;} Selects more than one type for styling
inside space div h2 {color: red;} Selects elements within other elements
tag class tag. h3.classname {color: red;} Selects the element with specified tags of specified class
Parent > p > a {color: red;} Selects elements that have other specified elements as parents
First After + h3 p {color: red} Selects the first elements that are immediately after another specified element
All After ~ th ~ td {color: red;} Selects all elements immediately after specified element
